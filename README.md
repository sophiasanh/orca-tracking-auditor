# 🐋 Orca Tracking Auditor

Ecommerce tracking audit tools built in Streamlit — analyzing attribution, UTM hygiene, spend efficiency, and conversion discrepancies across Shopify, Meta Ads, and Google Ads.

## Features
- 3-way attribution comparison — Shopify vs Meta vs Google
- Revenue discrepancy detection across platforms
- UTM hygiene checks — missing UTMs, casing inconsistencies, campaign name mismatches
- Spend efficiency alerts — campaigns spending $200+ with zero conversions
- High click / low conversion anomaly detection
- Tracking Health Score out of 100
- AI Executive Summary via Anthropic Claude API (optional)

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
3. Open http://localhost:8501

Or click **Load Sample Data** in the sidebar to run a demo instantly.
